## To create a new post:

1. Create a new file in `_posts` folder: `YYYY-MM-DD-title.md`
2. The template of the newly created post is:
```
---
layout: post
title:  "Event#12 Quarkus"
image: /path/to/post_image.png
date:   YYYY-MM-DD HH:MM:ss +0200
---
(excerpt content)
<!--more-->
(additional content)
```
